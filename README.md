# hassio

Minhas configurações pessoais utilizadas no Home Assistant em casa e no trabalho

# Software:

- Hassio (HassOS)
- Zigbee2MQTT
- SAMBA share
- FTP
- MQTT Broker
- Duck DNS
- SSH & Web Terminal
- NGINX
- TasmoAdmin
- Configurator
- AirCast
- HomeKit (Siri)

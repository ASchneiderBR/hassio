- Zigbee2MQTT
- SAMBA share
- FTP
- MQTT Broker
- Duck DNS
- SSH & Web Terminal
- NGINX
- Configurator
- AirCast
- HomeKit (Siri)
- Plex Media Server
- Pi-Hole (OrangePi)
- ESPHome
